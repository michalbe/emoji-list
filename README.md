# emoji-list by [@michalbe](http://github.com/michalbe) #
Array of 663 emoji icons

### How to use: ###
```
npm install emoji-list
```
then:
```javascript
var el = require('emoji-list');

console.log(el);
// [ ':bowtie:',
//  ':smile:',
//  ':laughing:',
//  ':blush:',
//  ':smiley:',
//  ':relaxed:',
//  ':smirk:',
//  ':heart_eyes:',
//  ':kissing_heart:',
//  ':kissing_closed_eyes:',
//  ':flushed:',
//  ...

```
